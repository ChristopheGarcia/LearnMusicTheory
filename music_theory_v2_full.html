Guide de démarrage — FL Studio & Rekordbox
FL STUDIOREKORDBOX▾
FL Studio = Image-Line · Version gratuite (trial) disponible · Achat unique, mises à jour à vie
PREMIER DÉMARRAGE — EN 10 ÉTAPES
1
Installer & lancer
Télécharge FL Studio sur image-line.com. À l'ouverture, charge un template vide (New → Empty). Configure ton interface audio dans Options → Audio Settings.
2
Comprendre l'interface
4 fenêtres clés : Channel Rack (tes sons, F6) · Piano Roll (mélodie, F7) · Mixer (effets, F9) · Playlist (arrangement, F5). Garde-les en mémoire.
3
Ton premier beat
Dans le Channel Rack, clique + → Add Beat+Bassline. Choisis tes sons (Kick, Snare, HH). Active les cases dans le séquenceur. Space pour écouter.
4
Définir le BPM
En haut à gauche du toolbar : le compteur BPM. Clique-glisse pour changer. Ou tap tempo avec le bouton Tap. Commence à 90 BPM pour un beat hip-hop classique.
5
Ajouter une mélodie (Piano Roll)
Ajoute un instrument virtuel (ex: PoiZone ou 3xOSC). Double-clique sur une case de la Playlist → Piano Roll s'ouvre. Dessine tes notes à la souris. Quantize avec Ctrl+Q.
6
Samples & Sampleurs
Glisse un fichier audio (.wav, .mp3) depuis le Browser vers la Playlist ou un canal. Utilise Fruity Granulizer ou DirectWave pour manipuler les samples de façon créative.
7
La basse (808 / Sub)
Charge GMS ou un plugin 808. Dans le Piano Roll, dessine une note longue. Utilise le portamento (glide) entre les notes pour l'effet 808 glissant typique du trap.
8
Mixer & effets
Assigne chaque canal au Mixer (clic droit → Assign to mixer track). Ajoute Parametric EQ 2 sur chaque piste. Règle les fréquences : coupe le bas sur hi-hat, boost mid sur snare.
9
Arrangement
Dans la Playlist, place tes patterns en blocs. Crée variation A (verse) et variation B (hook). Alterne-les. Ajoute des fills (1–2 mesures) aux transitions. Pense en multiples de 4 mesures.
10
Export
File → Export → WAV. Règle sur 320kbps MP3 ou WAV 24bit. Pour le partage : MP3. Pour le mastering : WAV. Coche "Split mixer tracks" pour envoyer tes stems à un ingé son.

Rekordbox = logiciel officiel Pioneer DJ · Mode Performance gratuit · Export USB payant (ou abonnement)
PRÉPARER SES TRACKS — WORKFLOW COMPLET
1
Importer ta musique
Glisse tes fichiers audio dans Rekordbox. Il supporte MP3, FLAC, WAV, AIFF. Évite les MP3 < 320kbps — artefacts audibles sur gros systèmes. Organise en playlists par genre ou BPM.
2
Analyse automatique
Clic droit → Analyze Track (ou analyse en masse). Rekordbox détecte BPM, grille rythmique, tonalité (Camelot). Vérifie toujours la grille manuellement — les détections sont parfois fausses.
3
Corriger la grille rythmique
Clique sur la waveform pour ouvrir le Beat Grid Editor. Si la grille est décalée, utilise Slip Grid ou Double/Half BPM. Une grille correcte = sync parfait avec les autres platines.
4
Placer les Hot Cues
Les Hot Cues (touches A–H) marquent les points importants. Place-en sur : intro, drop, break, dernier drop. Convention : Cue A = début du drop. Appuie sur un pad pour y sauter instantanément.
5
Memory Cues & Loops
Les Memory Cues apparaissent dans la waveform comme repères visuels. Les Loops permettent de boucler une section (ex : 4 mesures du drop pendant que tu prépares le prochain titre).
6
Harmonic Mixing — Clé Camelot
Rekordbox affiche la clé Camelot de chaque titre. Active le filtre par clé dans la bibliothèque. Pour mixer harmoniquement : reste sur la même clé ou passe à ±1. Utilise Key Filter pour trouver les tracks compatibles.
7
Exporter sur USB
Branche une clé USB formatée FAT32 ou HFS+. Dans Rekordbox : File → Export to Device. Sélectionne tes playlists. Les CDJ/XDJ de Pioneer liront ta bibliothèque avec toutes les données (cues, grilles, key).
8
Mode Performance
Avec une interface DJ connectée (ou Pioneer DDJ), passe en mode Performance. Tes decks A et B sont prêtes. Utilise le EQ 3 bandes pour les transitions : coupe les basses du track entrant avant de les remonter progressivement.
🎯 TECHNIQUES DE MIXAGE ESSENTIELLES
  • EQ Mix : coupe les basses du track B, booste sur A → inversé quand le drop arrive
  • Filter Transition : passe un filtre passe-haut sur le track sortant pendant la transition
  • Loop out : loupe 8 mesures du track sortant pendant que tu cherches le bon moment
  • Phrase mixing : mixe toujours au début d'une phrase de 4 ou 8 mesures — jamais au milieu
  • Gain matching : avant de mixer, égalise les niveaux — les VU-mètres à la même hauteur
